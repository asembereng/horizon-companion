Chapters 4 & 5: Clean Code & Design Principles

horizon/core/models.py · horizon/core/services.py

Dictionaries become proper domain models with dataclasses, enums, and SOLID principles. The Repository pattern and dependency injection decouple storage from logic.

Python models.py
@dataclass
class Task:
    id: int
    title: str
    assignee: User
    status: TaskStatus = TaskStatus.TO_DO
    priority: TaskPriority = TaskPriority.NORMAL

    def can_be_deleted_by(self, user: User) -> bool:
        return user.id == self.assignee.id

class TaskRepository(Protocol):
    # Abstract interface — depend on this, not a concrete DB
    def save(self, task: Task) -> Task: ...
    def find_by_id(self, task_id: int) -> Task | None: ...
    def delete(self, task_id: int) -> bool: ...
Python services.py — Dependency Injection
class TaskService:
    def __init__(self, repo: TaskRepository, notifier: NotificationService | None = None):
        self._repo = repo        # Could be in-memory or SQL
        self._notifier = notifier # Could be console or email

    def delete_task(self, task_id: int, requesting_user: User) -> bool:
        task = self._repo.find_by_id(task_id)
        if not task.can_be_deleted_by(requesting_user):
            raise PermissionError("Not authorized")
        return self._repo.delete(task_id)
Key insight TaskService takes repo as a parameter. In tests, pass InMemoryTaskRepository. In production, pass SQLiteTaskRepository. The service never knows or cares which one.
Dataclasses Enums SRP OCP DIP Protocol Repository pattern

Chapter 10: Security

horizon/core/auth.py · tests/test_ch10_security.py

Password hashing with bcrypt (rounds=12), HMAC token generation, input validation, and XSS sanitization.

Python auth.py
def verify_token(token: str) -> dict | None:
    parts = token.split(".")
    if len(parts) != 3: return None

    header_b64, payload_b64, signature = parts
    expected = hmac.new(_SECRET_KEY.encode(),
        f"{header_b64}.{payload_b64}".encode(), hashlib.sha256).hexdigest()

    if not hmac.compare_digest(signature, expected):
        return None  # Tampered
Production note This is a simplified JWT for learning. In production, use PyJWT or python-jose. The concepts (signing, verification, expiration) are identical.

    CMD python -c "import urllib.request; urllib.request.urlopen('http://localhost:8000/health')"

class TaskViewSet(viewsets.ModelViewSet):
    queryset = Task.objects.all()
    serializer_class = TaskSerializer
    permission_classes = [IsAuthenticated, IsAssigneeOrProjectAdmin]

    filterset_fields = ["status", "assignee"]

Python permissions.py
class IsAssigneeOrProjectAdmin(permissions.BasePermission):
    def has_object_permission(self, request, view, obj):
        if request.method in permissions.SAFE_METHODS:
            return True
        return (obj.assignee == request.user
                or obj.project.admin == request.user)

Chapter 6 (Django): Atomic Transactions

horizon/django/tasks/transactions.py

Database transactions with transaction.atomic() and select_for_update() to prevent race conditions when completing tasks.

Python transactions.py
from django.db import transaction

def complete_task(task_id: int) -> Task:
    with transaction.atomic():
        task = Task.objects.select_for_update().get(id=task_id)
        task.complete()
        task.save()
        return task
